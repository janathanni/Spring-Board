# Spring-Board
스프링 프레임워크를 공부하기 위해 만들어본 게시판입니다. <br>
구현한 기능은 다음과 같습니다. <br>
회원가입, 로그인, 게시판 작성/조회/수정/삭제, 댓글 작성/조회/수정/삭제, 좋아요/싫어요
